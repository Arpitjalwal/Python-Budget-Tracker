Personal Budget Tracker 📊
Project Overview
Ye ek Python-based financial management application hai jo users ko unka budget manage karne mein madad karti hai. Is project mein maine Object-Oriented Programming (OOP) ka use karke alag-alag categories (jaise Food, Clothing, Auto) ke liye ledger maintain kiya hai.

Key Features
Deposit & Withdrawal: Har category mein amount add karne aur nikalne ki suvidha, saath hi description add karne ka option.
Fund Transfer: Ek category se dusri category mein asani se funds transfer karne ka logic.
Balance Validation: Withdrawal ya transfer karne se pehle funds check karne ka mechanism taaki balance negative na ho.
Spending Analysis (Bar Chart): Ek custom function jo saari categories ke kharchon ko analyze karke ek percentage-based bar chart generate karta hai.

Technical Skills Demonstrated
Python Classes & Objects: Data ko organize karne ke liye Category class ka use kiya.
String Formatting: Terminal par ek clean aur aligned output (stars, categories, aur chart) dikhane ke liye advanced string formatting ka use kiya gaya hai.
Algorithmic Logic: Categories ke naam ko vertically print karne aur spending percentage calculate karne ka logic build kiya.

How to Use :
